# 歐付寶 全方位(All In One)電子發票介接Java版
---

## 1. 介紹

  - 歐付寶對於有開立電子發票需求的會員，提供完整的電子發票開立介接API，並有多種功能可選擇使用。 本套件為Java版，可使用開立發票，發送開立發票通知，查詢發票明細等應用。
  - 開立發票：
    - 一般開立發票
    - 延遲開立發票
    - 暫存開立發票
    - 開立發票折讓
    - 作廢發票
    - 作廢發票折讓


## 2. 安裝環境
  - _jdk 1.6以上_
